# Security Policy

## Supported Versions

The following versions receive security updates:

| Version | Supported |
| ------- | --------- |
| 5.1.x   | ✅        |
| 5.0.x   | ❌        |
| 4.0.x   | ✅        |
| < 4.0   | ❌        |

## Reporting a Vulnerability

Please report suspected vulnerabilities by opening a private security issue or emailing the maintainers. You can expect an initial response within 5 business days, followed by weekly status updates. Accepted reports will receive a coordinated fix and disclosure timeline; declined reports will include a brief rationale.
